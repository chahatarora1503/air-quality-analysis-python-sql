# 🌍 Urban Air Quality Analysis (Python + SQL)

## 📌 Project Overview
This project analyzes air quality data across major Indian cities to identify pollution patterns, seasonal trends, and key contributing factors.

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL (SQLite)
- Google Colab

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- AQI category distribution analysis
- City-wise pollution analysis
- Monthly trend analysis
- Pollutant correlation analysis
- City risk segmentation

## 🧠 Key Insights
- Majority of air quality falls under Moderate category
- Delhi and Gurugram show highest pollution levels
- Pollution peaks during winter months (Nov–Jan)
- PM2.5 is the most critical pollutant affecting AQI
- Southern and northeastern cities show better air quality

## 🗄 SQL Usage
- Aggregated AQI categories
- Identified top polluted cities
- Analyzed monthly pollution trends

## 📌 Conclusion
Air pollution in India shows strong seasonal and regional patterns. Targeted policies are required to reduce pollution in high-risk urban areas.
